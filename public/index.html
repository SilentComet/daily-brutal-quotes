iPhone Shortcut Setup

Automate your lock screen — runs daily, zero effort.

  1. 1
    Open the Shortcuts app on your iPhone and tap + to create a new shortcut.
  2. 2
    Add action: Get Contents of URL
    Set the URL to:
    https://your-domain.vercel.app/api/image?theme=neo
  3. 3
    Add action: Set Wallpaper
    Input: result from step 2 · Position: Lock Screen
  4. 4
    Tap the shortcut name → Automation → Add automation trigger.
    Set it to run Every Day at your preferred time (e.g., 7:00 AM).
  5. 5
    Disable "Ask Before Running" so it runs silently in the background.
